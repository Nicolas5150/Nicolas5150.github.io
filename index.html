Similar Dish

Source File Live Demo
Created with, Claraifi and Spoonacular API.

Website built on Google's Materialize framework.
The website takes in a user's image, uploads it, then creates a related url which is sent to Claraifi. There it processes the image and returns any ingredients it believes are 85% possible of being in the image. This list is sent to Spoonacular and returns any meals that can be made using the given ingredients. It is then presented to the user; clicking reveals the needed and given ingredients - as well as the cooking instructions if possible.

Time and Distance

Source File Live Demo
Created with Materialize, and Google Maps API.

Website built on Google's Materialize framework.
The website takes in a user's current location and destination (using googles places API). It processes the two waypoints (using Google's Distance Matrix). After it's calculated, it notifies the user of trip length and when they should depart their location. Once done they can text themselves, which includes a link to a Google Maps route.

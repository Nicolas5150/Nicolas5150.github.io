Line Waiting Concept

Source File Live Demo
Created with PHP and Google Maps API.

An in progress idea that I started at a Hackathon and been working on - as a side project. The navigation and walk through concept have been created in Sketch for iOS.
Currently I am working on a functional (non graphical) prototype for the web.
So far, using phpMyAdmin as a database for new users, and Google Map’s API, a user can create a new account and then pin on the map a set of coordinates, which is sent back to the database via an ajax call.
The concept of the site is to allow users to set a location for others to see (line waiters). These line waiters would wait on behalf of the other person in exchange for money.

Source File
